Deploy to GitHub / The World!

To upload stuff to GitHub you’ll need to tell GitHub that your computer is allowed to do that.

  1. start Hexo’s git-deployer installing

  2. Install Git’s offical version of GitForWindoze.

    • on the third page select “Use Git from the Windows Command Prompt”
    • on the fourth page select “Checkout as-is, Commit as-is”

  3. Right click on … somewhere … and check for an existing identity;

    • if either of these fail - you have no identity and need to make one up by typing

      … luckily there’s nothing to save

    • if both of those work - you can paste the clipboard contents into GitHub’s SSH keys section
  4. Rock-and-Roll ; go to the folder that has the site and generate the HTML, then deploy it all!
